# Finance x Technology Portfolio

Professional portfolio website for a University of Waterloo finance and technology student, built with HTML, Tailwind CSS, and minimal JavaScript.

## Pages

- `index.html` - landing page, project grid, resume section, and contact section
- `projects/risk-analysis.html` - sample risk analysis project page
- `projects/stock-pitch.html` - sample stock pitch project page

## Deployment

This site is ready to deploy on GitHub Pages, Vercel, or any static hosting platform.
